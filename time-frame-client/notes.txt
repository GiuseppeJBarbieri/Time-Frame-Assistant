'/':
GET - Retrieve all objects
POST - Creating object
PUT/PATCH - Update object by ID (can also give redundant data that wont change...)
DELETE - Deletes object by ID

'/:id':
GET - Retrieves object by ID

'/by<column_name>/:<column_name>':
POST - Retrieves one or all object(s) where x = x, y = y, ...
